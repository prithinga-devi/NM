# Assets Directory

This directory is for project assets such as:

- Images for documentation
- Icons and logos
- Sample equation images for testing
- Screenshots for README

## Structure

```
assets/
├── sample_equations/        # Sample equation images for testing
├── screenshots/             # UI screenshots for documentation
├── icons/                   # Application icons
└── readme_images/           # Images used in documentation
```

## Usage

Place any static files here and reference them in documentation as:
```markdown
![Description](assets/filename.ext)
```

## Contributing Assets

When adding assets:
1. Use descriptive filenames
2. Organize by category (subdirectories above)
3. Keep file sizes reasonable (compress images)
4. Update this README if adding new categories
